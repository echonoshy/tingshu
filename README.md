## Quick start

1. Install enviroments  
`.scripts/install.sh`

2. Download models weigths  
`./scripts/download_weights.sh`

3. Run web ui inferences  
`./scripts/run_gradio.sh`

4. Genereate audio data for frontend  
`./scripts/generate_web_data.sh`
